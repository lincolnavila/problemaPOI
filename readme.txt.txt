Para acessar o sistema, entre com a url:

- http://localhost:8080/problemaPOI/poi/index

Foi utilizado o PostgreSQL como banco para testes e afins de dados,
onde o hibernate utiliza o SCHEMA público e cria automaticamente sua tabela.
Também o tomcat 8.0 como servidor.

Foi utilizado os mesmo exemplos(base de dados) do enunciado do problema para confirmação dos testes.
